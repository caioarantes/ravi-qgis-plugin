What is RAVI?

The RAVI Plugin (Remote Analysis of Vegetation Indices) is a tool developed for QGIS that facilitates the calculation and analysis of vegetation indices using Google Earth Engine data. It allows processing satellite images to monitor vegetation health, agricultural productivity, and environmental changes.

Requirements for Use

  • An Active Google Earth Engine Account: A registered and approved GEE account (provides access to datasets, APIs, and tools).
  • A Google Cloud Project (and its ID): A Google Cloud project linked to your GEE account (handles authentication, API enablement, and quota management for Earth Engine services).
