O que é o RAVI?

O Plugin RAVI (Remote Analysis of Vegetation Indices) é uma ferramenta desenvolvida para o QGIS que facilita o cálculo e análise de índices de vegetação utilizando dados do Google Earth Engine. Ele permite processar imagens de satélite para monitorar a saúde da vegetação, produtividade agrícola e mudanças ambientais.

Requisitos para Uso

  • Uma Conta Ativa no Google Earth Engine: Uma conta GEE registrada e aprovada (fornece acesso a conjuntos de dados, APIs e ferramentas).
  • Um Projeto no Google Cloud (e seu ID): Um projeto do Google Cloud vinculado à sua conta GEE (gerencia autenticação, habilitação de API e gerenciamento de cotas para serviços do Earth Engine).
